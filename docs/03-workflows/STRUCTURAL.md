# Workflow: Structural Fabrication and Erection

## Material and member control

1. Receive structural shapes, plate, bar, grating, bolts, and consumables with grade, heat/lot, MTR/certification, coating requirements, and PMI when specified.
2. Create member/piece/assembly identities linked to drawing revision, sequence, WBS/work package, and source material genealogy.
3. Import or reference approved detailing/CNC data only through controlled revision and validation.

## Fabrication

Cut -> cope/drill/process -> fit-up -> welding/bolting preparation -> dimensional inspection -> NDE when specified -> assembly acceptance -> surface preparation/coating/galvanizing/fireproofing -> shipping release.

Track member mark, piece mark, assembly, heat, cut/remnant, CNC file revision, welds, holes, camber/sweep, coating, repairs, and inspection history.

## Erection

Foundation/anchor-bolt release -> delivery/receiving -> erection sequence -> temporary stability controls -> alignment/plumb/survey -> bolting and field welding -> coating touch-up/fireproofing -> punch -> final survey/acceptance.

## Bolting

Track connection, specification/method, bolt/nut/washer lots, storage, pre-installation verification, installation crew/date, turn-of-nut/calibrated wrench/TC/DTI or approved method, inspection, tension verification as required, and final acceptance.

## Controls

Use project-governed AWS/AISC/RCSC/ASTM/customer references without encoding unapproved acceptance rules. Member release requires current detailing, traceable required material, completed configured inspections, resolved holds/NCRs, and shipping/erection status authority.
